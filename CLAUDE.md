# Claude Code — Project Instructions

## Versioning

**Bump the minor version in `package.json` for every commit you create.**

- Format: `MAJOR.MINOR.PATCH` (e.g. `4.5.0` → `4.6.0`)
- Only the minor segment is incremented; patch and major are left unchanged unless explicitly requested.
- Include the version bump in the same commit as the feature/fix changes.
